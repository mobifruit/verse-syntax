## How You Can Contribute

You can help by:
- Adding or improving **Verse syntax highlighting rules**
- Extending **ANSI color themes** or Prism grammar
- Improving the **Discord bot functionality**
- Adding **documentation, examples, or guides**
- Reporting **bugs or issues** and suggesting improvements

## Getting Started

1. **Fork the repository**

2. **Clone your fork** locally:

`git clone https://github.com/YOUR_USERNAME/verse-syntax.git

3. **Create a feature branch**:

`git checkout -b feature/my-feature

4. Make your changes **following the coding style guidelines**

5. **Commit your changes** with a descriptive message:

`git commit -m "Add [feature/fix] description"

6. **Push your branch** to your fork:

`git push origin feature/my-feature

7. Open a **Pull Request** against the main repository

## Coding Guidelines
- Use **consistent indentation and spacing**
- Keep **function and variable names clear** and descriptive
- Comment complex logic for clarity
- Follow existing **file and naming conventions**

## Pull Requests
- Make sure your PR is **focused on a single change**
- Include **screenshots or examples** if relevant
- Link related issues using #issue_number
- PRs will be reviewed and **merged after approval** by the maintainers

## Reporting Issues

Please use the **GitHub Issues tracker** to report:

- Bugs in the bot or grammar
- Missing or incorrect syntax highlighting
- Feature requests or suggestions

Include **steps to reproduce** and any relevant examples to help maintainers understand the problem.
